# Placebo and falsification tests

Implemented development diagnostics are future-treatment placebo, pre-treatment outcome placebo, and deterministic treatment permutation. Apparent future/pre-treatment associations warn about leakage or residual confounding. Permutation does not prove identification.
# Python diagnostics

DoWhy placebo-treatment, random-common-cause, subset and bootstrap refuters are diagnostics. A noncontradictory result is not proof of causality or absence of confounding.
