# Pre-analysis checks

Thirty named gates cover source hash/schema/duplicates; validated dataset/features; analytical keys, calendar, launch timing, prescription/reimbursement policy, states and denominator; missing/suppressed/zero/Q4/censoring semantics; prohibited features, leakage and market-weight windows; grouped chronological split; causal timing/availability/domains/support; Python lock and contract; documentation; Git data safety; and secret-pattern metadata. Blocking findings prevent approval. Warnings remain in the package.
