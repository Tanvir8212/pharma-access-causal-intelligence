# State-entry policy

Supported policies are `AnyPositiveUtilization`, configurable `MinimumPrescriptionThreshold`, `PositivePrescriptionAndReimbursement`, and configurable `ConsecutiveQuarterConfirmation`. Each returns presence, first-entry status, policy name, thresholds, reason, and warnings. Suppression is not treated as zero. State validity, scientific eligibility, and completeness are distinct; territories are not rejected by name.
